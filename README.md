## 搬瓦工是什么？先说清楚你在买什么

搬瓦工（BandwagonHost）是加拿大 IT7 Networks 旗下的 VPS 服务商，英文名字发音像"搬瓦工"，所以国内就这么叫开了。它主要卖的是 KVM 架构的虚拟私有服务器，简单说就是一台在海外机房里的虚拟机，你可以远程登进去跑各种服务。

## FAQ：搬瓦工常见问题

**Q：搬瓦工费用按月付还是年付划算？**
A：CN2 GIA-E 系列支持季付（$49.99/季）和年付（$169.99/年），年付相当于四折扣了一季，长期用年付更划算。KVM 和 CN2 基础款只支持年付。香港套餐只支持月付和年付。

**Q：搬瓦工服务器到期没续费会怎样？**
A：到期后 VPS 会被暂停，数据会保留一段时间，续费后可以恢复。具体保留时间取决于官方政策，建议提前续费避免数据丢失风险。

**Q：搬瓦工 IP 被封怎么办？**
A：可以在 KiwiVM 控制面板申请更换 IP，费用 $8 一次，算是偏贵的。所以如果业务有被封风险，建议买流量宽裕一点、机房多一点的 CN2 GIA-E 套餐，可以通过迁移机房变相换 IP。

**Q：搬瓦工和一般便宜 VPS 区别在哪里？**
A：主要差在线路。普通便宜 VPS 走国际公共线路，晚高峰容易堵；搬瓦工 CN2 GIA 走专属优化通道，稳定性有保障，在线率官方承诺 99.9%+。代价就是同配置下价格比纯价格导向的服务商贵一些。

**Q：买哪个套餐最合适？**
A：选择困难就买 CN2 GIA-E 标准款，$49.99/季或 $169.99/年，12 个以上机房可以随时切换，错了还能在 30 天内退款。
